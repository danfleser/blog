# Dev blog

Personal static blog using mdx, React and Next.js.
Can be used as a template.

# Install dependencies

```bash
npm install
```

# Run server locally

```bash
npm run dev
```

# Build for prod and export static website

```bash
npm run build
```

# Add a post

```bash
# run
npm run gen-post My new article

git add .
git commit -m "feat: new post"
git push
```
